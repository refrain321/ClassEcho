# ClassEcho

ClassEcho 是一个面向课堂学习场景的记录与复盘工具。
它可以帮助你在上课时保留关键信息，并在课后快速整理知识点与待解问题。

## 这个应用能做什么

- 课堂录音与文字记录
- 自动生成课程要点
- 标记课堂中的待解问题
- 在历史页继续查看和补充解答
- 插入白板照片，形成图文时间轴
- 导出复习卡片（Anki CSV）
- 网络波动时自动补偿失败片段

## 适合谁

- 在校学生
- 需要课后复盘的学习者
- 希望把课堂内容快速结构化的人

## 使用前准备

首次打开后，请在“设置”中完成以下项目：

- API Key
- 语音识别模型
- 总结模型
- Base URL（可选）

## 如何使用

1. 打开应用，进入首页。
2. 输入课程名称并开始录音。
3. 课堂进行中可随时拍摄白板图片。
4. 应用会持续生成要点，并支持把疑难点加入待解问题清单。
5. 结束后内容会自动保存到历史记录，方便课后复盘。

## 历史与导出

- 历史页可查看每节课的完整记录。
- 你可以在详情页补充问题解答。
- 可导出 Anki CSV 用于后续复习。

## 常见问题

1. 无法识别语音
- 检查麦克风权限
- 检查网络连接
- 检查 API Key 是否有效

2. 没有生成要点
- 确认模型配置正确
- 确认课堂内容长度足够

3. 离线期间数据异常
- 应用会缓存失败片段
- 网络恢复后可在历史页执行重试补偿
